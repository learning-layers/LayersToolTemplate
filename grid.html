Using grid for UI layout

You can set up different sized blocks, as long as the sum is 12. 'sm' means that these columns start existing once the screen size is small or greater. Reduce the window size until you see the change.

Mixing different layout arrangements

If there are multiple column spans defined for one element, the largest possible will be used. This allows defining one area to span more columns as the screen gets smaller, good for video players, canvases or other hard-to-resize content. If size is smaller than smallest defined size, then each column takes its own row. (=col-xs-12)

.col-sm-8 .col-md-6 fixed size block
.col-sm-4 .col-md-8 Trying to use the remaining space.

The size tiers are: xs (phones / vertical small tablets), sm (tablets), md (desktops), and lg (larger desktops).
